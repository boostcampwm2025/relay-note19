#### 1. 첫번째 퀘스트
```go
퀘스트: 게임에서 시간이 다 되면 게임오버가 되듯이 네이버 부스트캠프 챌린지 과정을 진행해보자

목적: 시간 안에 최선의 퀄리티를 내는 연습
1. 네이버 부스트캠프 주간 일정 시간표와 추가적으로 자신이 지키고 싶은 시간대들을 AI에게 학습시키기
2. 아래 예시와 같이 일주일 시간표를 정하고 학습시키기
    ex) 화: 10-11 피어 컴파일링
            11-12 피어 피드백
            12-14 README.md, 나만의체크포인트.md 초안 작성 및 gist제출
            14-18 문제 해결과 학습
            18-19 나만의 체크포인 점검
            19-20 체크아웃 및 길잡이 자료
            20-24 추가 학습
3. 각 일정의 마감 시간 10분전 AI가 알려주고 해당 작업을 진행 상황까지 마무리 작업을 같이 도와줌
4. 마무리 작업을 도와주는 기준점은 정하기 나름대로
    ex) 어떤 코드를 작성중이였다면 23:50에는 마무리하고 자야하므로
    구현중이던 코드는 빌드가 성공하는 정도로 마무리한다던지 
    or 구현 못한 부분들을 AI가 작성한 표시와 함께 마저 구현한다던지
    
    +README.md, 나만의체크포인트.md도 위와 같이 마무리의 기준을 정해 마무리를 도와주도록 설계하기
5. 최소한의 달성 기준: AI에게 시간표를 학습시키고 각 일정의 10분전에 알림을 보내고 확인할 수 있기
                    알림은 자신이 확인할 수만 있다면 어떤 형태로 제공해도 좋다.
```

```go
최소한의 달성 기준: 수면 분석 및 이후 하루 컨디션 관리를 위한 방법을 얻는 프롬프트를 작성한다.

(선택)추가 기준: 만약 이 작업을 매일 아침 자동화할 수 있다면 자동화를 해보자.
예를 들어 스마트 워치의 데이터를 자동으로 가져와서 기존에 설정한 프롬프트 템플릿에 입력해서
자동으로 AI에 질문하고 답을 휴대폰으로 알림으로 보내준다 등등
```

- 퀘스트 수행 결과
- 해당 퀘스트를 수행하는 데 AI는 어떤 역할이었나요? AI가 없으면 동일한 결과를 얻을 수 없을까요?

- 퀘스트 수행 결과


**작성 프롬프트 예시**

기상시간, 취침시간, 그리고 나의 기분을 토대로 오늘 점심 메뉴를 추천해줘

---

월요일 기상시간: 8시10분
월요일 취침시간: 새벽 2시
기분: 주말에 푹 쉬어서 그런지 일어날 때 개운했다. 잠을 많이 잔 것 같다.
점심 메뉴:
- 연어 샐러드 정식 (샐러드 + 구운 연어 + 현미밥): 연어의 오메가-3는 뇌 기능을 활성화하고, 신선한 채소는 가벼운 소화를 도와 개운한 기분을 유지해줍니다. 현미밥은 지속적인 에너지 공급에 도움.
- 음료: 자몽 주스 (비타민 C로 상쾌함 유지).
- 대안 메뉴: 닭가슴살 퀴노아 샐러드 또는 비빔밥 (채소와 단백질 균형).


화요일 기상시간 8시분
화요일 취침시간: 새벽 1시
기분: 전날에 미션을 하루종일 진행하느라 역시나 힘들었다. 몸이 매우 뻐근하고 머리가 잘 안 돌아가는 것 같다.
점심 메뉴:
- 소고기 미역국 정식 (미역국 + 소고기 + 잡곡밥 + 반찬): 미역국은 따뜻하고 소화가 쉬워 몸을 편안하게 해주며, 소고기의 단백질과 철분은 에너지 회복에 도움. 잡곡밥은 피로 회복을 위한 복합 탄수화물 제공.
- 음료: 따뜻한 허브차 (예: 캐모마일, 긴장 완화와 기분 전환에 도움).
- 대안 메뉴: 닭곰탕 또는 콩나물국밥 (따뜻하고 부드러운 국물 요리로 위에 부담 적음).
